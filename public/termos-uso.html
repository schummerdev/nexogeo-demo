⚖️ Termos de Uso

Última atualização: 15 de setembro de 2025

📋 Resumo Executivo: Estes termos regem o uso do sistema NexoGeo2 para gestão de promoções, definindo direitos, deveres e responsabilidades de usuários e administradores.

1. 📝 Aceitação dos Termos

Ao utilizar o sistema NexoGeo2, você concorda integralmente com estes termos. Caso não concorde, deve interromper imediatamente o uso do sistema.

2. 🎯 Finalidade do Sistema

O NexoGeo2 é um sistema de gestão de promoções que permite:

  • Cadastro de participantes em promoções
  • Controle de participações
  • Realização de sorteios transparentes
  • Gestão de ganhadores e prêmios
  • Auditoria completa de todas as operações

3. 👤 Tipos de Usuário

Participantes:

  • Podem se cadastrar em promoções ativas
  • Devem fornecer dados verdadeiros e atualizados
  • Têm direito a transparência nos sorteios

Administradores:

  • Podem gerenciar promoções e participantes
  • Têm acesso a relatórios e auditoria
  • São responsáveis pela operação correta do sistema

4. 📋 Responsabilidades dos Usuários

Participantes devem:

  • Fornecer informações verdadeiras e atualizadas
  • Não criar múltiplas contas
  • Respeitar as regras específicas de cada promoção
  • Não tentar burlar o sistema

Administradores devem:

  • Usar o sistema apenas para fins legítimos
  • Manter credenciais de acesso seguras
  • Respeitar a privacidade dos participantes
  • Seguir os procedimentos de auditoria

5. ⛔ Condutas Proibidas

⚠️ É expressamente proibido:
  • Fornecer dados falsos ou de terceiros
  • Tentar acessar o sistema sem autorização
  • Interferir no funcionamento normal do sistema
  • Criar múltiplas contas para uma mesma pessoa
  • Usar bots ou scripts automatizados
  • Violar direitos de propriedade intelectual

6. 🛡️ Segurança e Privacidade

  • Todos os dados são tratados conforme nossa Política de Privacidade
  • O sistema implementa medidas de segurança adequadas
  • Logs de auditoria registram todas as ações
  • Senhas são armazenadas de forma criptografada

7. 🎲 Sorteios e Transparência

  • Todos os sorteios são realizados de forma transparente
  • O sistema utiliza algoritmos verificáveis
  • Logs completos são mantidos para auditoria
  • Participantes podem solicitar informações sobre sorteios

8. 📊 Propriedade Intelectual

  • O sistema NexoGeo2 é propriedade de seus desenvolvedores
  • Usuários não podem copiar, modificar ou redistribuir o código
  • Dados inseridos pelos usuários permanecem de sua propriedade

9. 🚫 Limitação de Responsabilidade

  • O sistema é fornecido "como está"
  • Não garantimos disponibilidade 100% do tempo
  • Usuários são responsáveis por backup de seus dados importantes
  • Em caso de problemas técnicos, faremos o melhor para resolver rapidamente

Estes termos podem ser modificados a qualquer momento. Mudanças significativas serão comunicadas através do sistema e por e-mail.
